# Функция экспорта отгрузочных листов в PDF

## Описание
Добавлена возможность скачивания отгрузочных листов в формате PDF (А4) для удобства печати и документооборота.

## Возможности
- ✅ Экспорт в PDF формате А4
- ✅ Автоматическое разбиение на страницы при необходимости
- ✅ Включение всей информации об отгрузке
- ✅ Таблицы для оборудования и аренды
- ✅ Чек-лист с отметками
- ✅ Места для подписей
- ✅ Русская локализация

## Где доступна функция

### 1. В списке отгрузок
- Кнопка "PDF" рядом с кнопками "Просмотр" и "Редактировать"
- Быстрый доступ к скачиванию без открытия деталей

### 2. В модальном окне просмотра отгрузки
- Кнопка "PDF" в правом верхнем углу заголовка
- Доступ к полной информации об отгрузке

### 3. В форме редактирования отгрузки
- Кнопка "PDF" в правом верхнем углу заголовка
- Только при редактировании существующей отгрузки

### 4. В чек-листе отгрузки
- Кнопка "PDF" в заголовке чек-листа
- При наличии данных об отгрузке

## Содержимое PDF

### Заголовок
- Название документа: "ОТГРУЗОЧНЫЙ ЛИСТ"
- Номер отгрузки
- Дата создания документа

### Основная информация
- Дата отгрузки
- Статус отгрузки
- Ответственное лицо
- Общее количество позиций

### Получатель
- Наименование получателя
- Адрес доставки

### Комментарии
- Дополнительные комментарии (если есть)

### Оборудование
- Таблица с колонками: №, Наименование, Серийный номер, Количество, Отметка
- Автоматическое разбиение на страницы при большом количестве позиций

### Стеки техники
- Список стеков с указанием количества
- Детальный состав каждого стека
- Информация об оборудовании в стеке

### Арендованное оборудование
- Таблица с колонками: №, Наименование, Количество, Отметка

### Чек-лист
- Список пунктов для проверки
- Отметки о выполнении (☑ для выполненных, □ для невыполненных)
- Описания пунктов (если есть)

### Подписи
- Линии для подписей отправителя, водителя и получателя
- Дата и время создания документа

## Технические детали

### Используемые библиотеки
- `jspdf` - основная библиотека для генерации PDF
- `jspdf-autotable` - для создания таблиц

### Формат
- Размер страницы: А4 (210 x 297 мм)
- Ориентация: портретная
- Шрифт: Helvetica (стандартный для PDF)
- Кодировка: UTF-8

### Автоматическое разбиение
- Контроль размера содержимого на странице
- Автоматическое добавление новых страниц при необходимости
- Сохранение читаемости и структуры документа

## Использование

### Скачивание PDF
1. Найти нужную отгрузку в списке
2. Нажать кнопку "PDF" (с иконками Download и FileText)
3. Дождаться генерации документа
4. Файл автоматически скачается с именем: `Отгрузочный_лист_НОМЕР_ДАТА.pdf`

### Именование файлов
Формат имени файла: `Отгрузочный_лист_{номер}_{дата}.pdf`
Пример: `Отгрузочный_лист_SH-2024123456_25-12-2024.pdf`

## Преимущества

1. **Удобство печати** - документ готов для печати на стандартной бумаге А4
2. **Портативность** - можно отправить по email или сохранить в облаке
3. **Официальность** - документ содержит все необходимые поля для подписей
4. **Полнота информации** - включает все данные об отгрузке
5. **Автоматизация** - не требует ручного создания документов

## Совместимость
- Работает во всех современных браузерах
- Поддерживает печать на любых принтерах
- Совместим с системами документооборота
- Можно открывать в любых PDF-ридерах
